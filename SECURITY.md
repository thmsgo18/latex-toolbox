# Security Policy

## Supported Versions

Only the latest version published on [PyPI](https://pypi.org/project/latex-forge/)
is supported with security fixes. Please upgrade before reporting an issue:

```bash
pipx upgrade latex-forge
```

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Instead, use GitHub's [private vulnerability reporting](https://github.com/thmsgo18/latex-forge/security/advisories/new),
or email thomas.gourmelen@icloud.com with details and steps to reproduce.

You should expect an initial response within a few days. If the issue is
confirmed, a fix will be released as soon as possible and credited in the
[changelog](CHANGELOG.md) unless you prefer to remain anonymous.
