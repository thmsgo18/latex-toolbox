Ce dossier peut servir a stocker des projets reels si tu veux les garder dans la toolbox.

La commande `latex-forge` ne cree plus automatiquement ici : elle cree un projet dans le dossier courant.

Exemples :

```bash
cd /path/to/latex-forge/projects
latex-forge create --name projet-big-data --template rapport-projet-en
latex-forge create --name rapport-ter-vision --template rapport-ter
latex-forge create --name paper-audio-search --template research
```
