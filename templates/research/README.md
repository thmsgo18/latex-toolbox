Research template in English based on the layout of `main-12.tex`.

Default structure:

- title page
- abstract
- table of contents
- two-column main body
- introduction
- conclusion
- appendix with AI usage note
- bibliography based on `references/references.bib`

Notes:

- the title page uses the same structure as the project-report templates
- metadata are centralized in `frontmatter/metadata.tex`
- the abstract is isolated in `frontmatter/abstract.tex`
- references are managed through BibTeX with `references/references.bib`
- the template reuses the shared table, Python, and Bash styles
